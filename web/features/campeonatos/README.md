# Recurso de campeonatos

Componentes e lógica exclusivos de campeonatos.

- `fases/`: criação, edição e listagem de fases.
- `grupos/`: grupos vinculados a fases; slots são gerados automaticamente pelo backend.
- `jogos/`: jogos e vínculos entre jogos e grupos.
- `stream/`: aba Stream (hub de overlays + planilha fullscreen com dados reais; editor em rota própria).
- `components/`: cabeçalho, navegação e gerenciador da estrutura do campeonato.

Formulários globais de criação/edição de campeonato ficam em `web/components/forms/campeonato`.
